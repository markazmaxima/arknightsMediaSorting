Back to Scratch


I will be building the Archiver from scratch, from ground up in a new repository.

Old site will be stay as they are now but with a statement on front page, link to the announcement page and link to the new main page.



User Interface


Mobile UI will not be the focus. You may navigate the website from mobile but it won't be optimize anytime soon.

Desktop UI will be the focus.


UI will be more for archiving and organizing purpose and may not be friendly for new player.



Layout re-Design


No more experimental style, I will design it like how an Arknights developer would design.



Files & Folders


Previously I separate multiple page with multiple css jQuery and JSON for each page.

Now I decide to make "giant" files to hold duplicated or formatted elements to reduce self confusion.


Moreover, the usage of iFrame will be reduced drastically and replace with hard code it together with the Parent layer or embed it. Gonna see which one work well.


Previous version of navigation system which utilize iFrame will be adopted to this brand new website.



Content
